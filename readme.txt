1.Outline
This project is to analyze three types of data:
    (1). Statistics Mesh Data for business activity
    (2). Statistics Mesh Data for Nationals
    (3). Covered area by co-ops' delivery service.

2.The workflow.
The process proceeds
    (1) Label each mesh with the level of Food Desert 
    (2) Convert the latitude of each mesh to zipcode
        For the conversion, I will use google API
	https://developers.google.com/maps/documentation/geocoding/intro?hl=ja
    (3) Check each zipcode whether it is covered by Co-ops' home delivery service.
